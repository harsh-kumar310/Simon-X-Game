# Simon X – Color Memory Game
A fun and interactive memory-based “Simon Says” color game built using HTML, CSS, and JavaScript .
Follow the color sequence, test your memory, and challenge yourself to beat your high score!

# Features
1.Beautiful color and dark themes
2.Username system with personalized high scores
3.Real-time leaderboard for all users
4.Data saved in browser using localStorage
5.Restart and play again feature
6.Smooth animations and modern UI design

# How to Play
1. Enter your name and click Start
2. Watch the color pattern carefully
3. Repeat the same pattern by clicking the colors
4. Each level adds one more color to the sequence
5. If you press the wrong color — game over!
6. Try to beat your highest score
# Tech
1. HTML5
2. CSS3 (Animations & Responsive Design)
3. JavaScript (DOM Manipulation & Logic)
